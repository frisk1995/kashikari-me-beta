旅行・飲み会・ルームシェアでの「立て替え」「割り勘」をかんたんに記録・精算できる iPhone アプリです。

■ こんな場面で活躍
・グループ旅行で1人がまとめて支払った宿泊費・交通費・食事代を後から精算したい
・飲み会やランチで誰かが代わりに払った費用をすっきり清算したい
・ルームシェアや彼氏彼女との日常的な立て替えを管理したい

■ 主な機能

【グループ管理】
旅行やイベントごとに「グループ」を作成。メンバーの名前を追加するだけですぐに使い始められます。

【支払い記録】
「誰が」「いくら」「何に」払ったかを記録。割り勘の対象メンバーを全員または個別に指定できます。カテゴリ（食事・交通・宿泊・買い物など）も設定可能。

【自動精算計算】
グループ内の貸し借りを集計し、「AさんがBさんに ¥X,XXX 払う」という形式で最小回数の精算案を自動表示。手計算は不要です。

【残高一覧】
各メンバーの払いすぎ・払い足りない金額を一目で確認できます。

【精算完了マーク】
精算が済んだ送金にチェックを付けて完了状態を管理。精算結果のテキスト共有・コピーにも対応。

【QRコードでメンバー招待・リアルタイム共有】
QRコードを表示してメンバーを招待するだけで、同一グループに自動参加。各自が自分の端末から支払いを記録でき、変更はリアルタイムで全員に反映されます。

■ シンプルで使いやすい設計
・アカウント登録不要 — 起動してすぐ使えます（匿名認証）
・データはクラウドに安全に保存。機種変更後も安心
・縦持ちスマホに最適化されたシンプルな操作性

■ プライバシーについて
本アプリは位置情報・広告ネットワーク等を使用しません。グループデータはクラウド（Firebase）に暗号化して保存されます。

割り勘・精算でもめるのはもう終わり。「Kashikari.me」でスッキリ精算しましょう。

■ プレミアムプラン（自動更新サブスクリプション）
・グループ無制限、メンバー無制限、テーマカラー全10色、CSVエクスポート
・年額プラン：¥2,800 / 年（月換算 ¥233）
・月額プラン：¥300 / 月
・購入確認時に Apple ID アカウントに請求されます
・期間終了日の24時間前までにキャンセルしない限り自動更新されます
・購入後は Apple ID のアカウント設定からサブスクリプションを管理・キャンセルできます

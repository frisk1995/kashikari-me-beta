10 実機テスト・サンドボックステスト

実機デプロイ

# USB接続後に実行
npx expo run:ios --device

# Metro をトンネルモードで起動（LAN接続問題を回避）
npx expo start --tunnel
注意: Xcodeから直接ビルドするとMetro URLがlocalhostになり実機から接続できない。 必ず npx expo run:ios --device を使うこと。

サンドボックステスト手順

  1. App Store Connect → ユーザーとアクセス → サンドボックステスター → テスターアカウント作成
  2. iPhone: 設定 → App Store → 一番下「サンドボックスアカウント」にサインイン
  3. アプリで課金ボタンをタップ → サンドボックス購入ダイアログを確認

サンドボックステストが動かない場合のチェックリスト

  • 有料アプリ契約が「有効」
  • RCのBundle IDが一致（com.kashikarime.ios）
  • RCにProduct ID登録済み
  • RCのOfferingsにPackage紐付け済み
  • サンドボックステスターアカウント設定済み
  • 有料アプリ契約反映待ち（最大数時間）

StoreKit Configuration ファイルを使ったローカルテスト

  1. Xcode で ios/StoreKit.storekit を作成
  2. Product ID: com.kashikarime.ios.premium.monthly を追加
  3. Scheme → Run → Options → StoreKit Configuration に設定
  4. Xcodeからビルドして実行

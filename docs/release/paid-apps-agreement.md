# App Store Connect 有料アプリ契約（Paid Apps Agreement）

## 概要

App内課金（サブスクリプション含む）を提供するには、無料アプリであっても
App Store Connect で **有料アプリ契約** を締結する必要がある。

未締結の場合、サンドボックス環境でも StoreKit がプロダクト情報を返さず、
RevenueCat の offerings が空になる。

---

## 締結手順

### 1. App Store Connect にアクセス

[https://appstoreconnect.apple.com](https://appstoreconnect.apple.com) にログイン後、
トップページの「契約/税金/口座情報」をクリック。

### 2. 契約タブを確認

「有料アプリ」行のステータスを確認する。

| ステータス | 意味 |
|-----------|------|
| 有効 | 締結済み（IAP 利用可能） |
| 処理中 | Apple が審査中（数日かかる場合あり） |
| アクション必要 | 情報の入力・署名が必要 |
| 新規 | 未締結（要手続き） |

「アクション必要」または「新規」の場合は以下の手順で締結する。

### 3. 銀行口座情報の入力

「口座情報」タブ → 「追加」

- 銀行名
- 支店名・支店コード
- 口座種別（普通 / 当座）
- 口座番号
- 口座名義（ローマ字）

日本の場合は国内銀行口座で受け取り可能。

### 4. 税務情報の入力

「税務情報」タブ → 日本居住者の場合：

- フォーム：**W-8BEN**（米国外個人向け）
- 氏名・住所・国籍・納税者番号（マイナンバーまたは TIN）を入力
- 源泉徴収率：日米租税条約により **0%** を申請可能

### 5. 契約への署名

「契約」タブ → 「有料アプリ」→「表示」→ 内容を確認して「同意する」

---

## 締結後の反映時間

| 項目 | 反映時間の目安 |
|------|-------------|
| 契約署名 | 即時〜数時間 |
| 銀行・税務情報の審査 | 1〜3 営業日 |
| サンドボックスでの IAP 動作 | 契約署名後、即時〜数時間 |
| 本番での IAP 動作 | アプリ審査通過後 |

---

## サンドボックステストとの関係

有料アプリ契約が **「有効」** でないと、サンドボックス環境でも
StoreKit（Apple の購買 API）がプロダクト情報を返さない。

RevenueCat で以下のエラーが出る場合は契約未締結を疑う：

```
RevenueCat.OfferingsManager.Error error 1
None of the products registered in the RevenueCat dashboard
could be fetched from App Store Connect
```

- [ ] 銀行口座情報が登録済み
- [ ] W-8BEN（税務情報）が提出済み
- [ ] サンドボックステスターアカウントが作成済み
- [ ] 実機の Apple ID をサインアウトし、購入時にサンドボックス ID でサインイン

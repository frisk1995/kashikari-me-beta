## 改善提案

- **削除後の遷移（バグ#1）**: 支払い削除は詳細画面に正しく戻るが、グループ削除は削除済み詳細画面に戻るため UX が片手落ち。`deleteGroup` 後はホーム（ルート）へ `router.dismissAll()` / `router.replace('/')` で戻すと一貫性が高まる。Sprint 5 のUX調整での対応候補。
- **金額ヒーロー入力の検証メモ**: 非数値入力は `onChangeText` の `[^0-9]` 除去で適切に弾けている。設計通り堅牢。
